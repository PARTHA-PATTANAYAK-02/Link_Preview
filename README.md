## 🧪 Getting Started

### 📦 Installation

Clone the repo :

```bash
git clone https://github.com/PARTHA-PATTANAYAK-02/Link_Preview.git
cd link_preview
```

install dependencies:

```bash
npm install
```

also see the package.json

Run the development server :

```bash
npm run dev
```
